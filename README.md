# Initial blueprint

* PWA Consecration reader.
* No build, no dependency vanilla JS PWA with ES modules.
* Built around ES module plugins.
* Beautiful CSS design with subtle animations on load and scroll.
* Track state locally, such as where in the Consecration the user is upto.
* Modular mode system. e.g. Traditional Luis de Montfort, Contemporary, Lite, St. Joseph etc.
* Catch up system for those that fall behind, giving a lite summary of missed days.
* Emoji button for night (🌛) and day (🌞) mode, starting with system default then overriding if changed.
* Web Audio API narration of core content.
* Accessibility at the forefront and built-in.
* Track streaks in a non-gamey kind of way, such as a discrete counter and optional calendar view.
* Suggested dates for consecrations, based on the liturgical calendar.
* Optional reminders. 🔔 emoji when activated.🔕 emoji when deactivated. Off by default.
* Option to download the PWA by clicking ⬇️ button.
* Modular JSON prayer files, called in at relevant times in the consecration.
* Integrate Logos Reftagger for Bible references.
