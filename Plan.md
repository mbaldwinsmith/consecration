# Consecration

* Progressive web app
* Based on St Louis de Montfort's consecration, updated
* Built in tracker, keeps track of day, progress, streak etc.
* Allows you to catch up on previous days
* Seperate markdown files for each recurring prayer, litany, hymn etc
* Mark down pages for each day's unique content
* Build step to bring together markdown files and convert them to HTML
* Capacity to make a St. Joseph's consecration mode (optional)
* Light and dark modes, defaulting to system default
* No flicker on load if light/dark mode different to system default
* Beautiful color palette and subtle CSS animations, including on scroll
* Not React dependant
* Options of various consecrations
* Stretch goal: include audio narration and/or videos
* Shareable social banners and posts to celebrate completing a day or milestones
