# Consecration

## Core Loop
1. Open app
2. See current day immediately
3. Read / listen
4. (Optional) reflect
5. Mark complete
6. Leave peacefully

## Modes
1. Guided (default)
    * Full content
    * Standard pacing
    * Gentle reminders

2. Lite
    * Shortened readings
    * Essential prayers only
    * Auto summaries for missed days

3. Custom
    * Rule-of-life editing
    * Pacing control
    * Optional extras
  
## Suggested plugin types:

* Content Packs
    - Consecrations (Marian, De Montfor, Joseph, etc.)
 
* Enhancements
    - Audio
    - Bible integration
    - Calendars
  
* UI Modes
    - Lite mode logic
    - Reflection/journal system

## Features
* Progressive web app
* Based on St Louis de Montfort's consecration, updated
* Built in tracker, keeps track of day, progress, streak etc.
* Allows you to catch up on previous days
* Seperate markdown files for each recurring prayer, litany, hymn etc
* Mark down pages for each day's unique content
* Build step to bring together markdown files and convert them to HTML
* Capacity to make a St. Joseph's consecration mode (optional)
* Light and dark modes, defaulting to system default
* No flicker on load if light/dark mode different to system default
* Beautiful color palette and subtle CSS animations, including on scroll
* Not React dependant
* Options of various consecrations at the end
* Front matter with e.g. Title, phase, author, duration, audio etc.
* Plugin based and extendable, rather than hard-coded for future proofing

* Allow sceen-off playback
* Lock screen controls
* Rule of life mode
* Lite mode
* Summarise missed days
* Helpful tracking. E.g. % completion, phase markers, badges, milestone reflections
* Generate a certificate PDF
* Timeline of the journey

## UX
* Swipe between days
* Long press to mark complete
* Tap sections to collapse/expand prayers
* No system UI
* Subtle animations
* Beautiful, contemplative colour pallette
* Offline first readability
* Frictionless daily use
* Choose time of reminders

### Catch-up Logic
* Day +1: simple “resume”
* Day +2–3 → offer:
    - “Catch up in one session”
    - “Summarise missed days”
* Day 4+ → suggest:
    - Lite mode temporarily

## Rules
* No cheap gamification
* No guilt or shame
* Avoid layering on pressure or overwhelm
